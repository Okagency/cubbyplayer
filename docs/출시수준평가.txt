큐비플레이어(CubbyPlayer) 출시 수준 평가
작성일: 2026/03/30

================================================================
1. 잘 갖춰진 부분
================================================================
- 핵심 기능 완성도 높음: 비디오 재생(ExoPlayer+VLC 듀얼엔진), 자막, 구간재생, 제스처 컨트롤
- 편집 기능: 분할/합치기/오디오 교체 (FFmpeg 기반)
- 캐스팅: Chromecast + DLNA 지원
- UI/UX: 다크 네이비 테마 일관성, Material Design, 한글 UI, 커스텀 앱 아이콘
- 안정성: CrashLogger, 재생 위치 저장(Room DB)
- 권한 처리: Android 13+ READ_MEDIA_VIDEO 대응
- ProGuard/R8: 모든 주요 라이브러리 규칙 적용 완료 (ExoPlayer, VLC, FFmpeg, Glide, Room, Coroutines, Cast, DataStore)

================================================================
2. 출시 전 보완 필요 사항
================================================================
[높음]
- 개인정보처리방침: Play Store 출시 시 필수 — 없음

[중간]
- ProgressDialog: android.app.ProgressDialog 사용 (API 26에서 deprecated)
- 테스트 부재: 유닛 테스트, UI 테스트 전무
- 에러 처리: PlayerActivity가 267KB로 거대 — 유지보수 리스크

[낮음]
- onBackPressed: deprecated API 사용 (OnBackPressedCallback 미사용)
- compileSdk 34: 최신은 35이나 34도 현재 출시 가능
- plugin-converter 모듈: settings.gradle에 포함되어 있으나 실체 미확인

================================================================
기능적으로는 출시 가능 수준.
비디오 플레이어 핵심 기능(재생, 자막, 제스처, 편집, 캐스팅) 모두 구현 완료.

- Play Store 출시: 개인정보처리방침 필수
